Twitter-Bot: The Twitter API Bot is a project that uses Twitter's API which enables us to play around a bit. In my case I created a automated reply system so that when a user follows me or tweets me there is an automated reply, taking in consideration whether they followed or @ mentioned my account. Their is also a search method implemented where you can search specific keywords that have been tweeted (through the terminal) Account: @VladAutomatic
